Maximum Likelihood Estimation (MLE) finds parameter values that make the observed data most probable. For Normal: μ̂ = sample mean, σ̂² = sample variance. For Exponential: λ̂ = 1 / sample mean.

The Log-Likelihood curve shows how likely different parameter values are — the peak represents the MLE.
